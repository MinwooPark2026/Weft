대본 작성 사람 Weft 범위 밖 · weft 명령 없음

사용자가 직접 쓴 산문 대본(.md)이 Weft의 입력. 주제는 자유 — 역사, 과학, 문화 분석, 사고실험, 철학 비교 등 무엇이든. "주제 → 대본" 단계는 Weft가 관여하지 않는다 (어떻게 쓰든 자유).

🤖 AI가 쓰이는 곳 (한눈에)

  • 대본(0) — AI 아님: 사용자가 직접 쓴다 (Weft 범위 밖, 주제 자유)
  • 콘티(1) — script-to-conti 스킬(Claude/LLM): 대본을 컷으로 쪼개고 shot별 영문 이미지 프롬프트·모션 작성 ← "대본을 콘티로 바꾸는" 핵심 AI 단계 (스킬 번들: .claude/skills/script-to-conti/)
  • 나레이션(3) — Typecast 신경망 TTS: 텍스트 → 음성
  • 이미지(4) — OpenAI gpt-image-1: 프롬프트 → 그림 (picker에서 추가 생성·프롬프트 수정도 AI)
  • 애니메이션(5) — AI 에이전트가 SPEC.md를 읽고 Remotion/HyperFrame clip 렌더 (해당 shot이 있을 때만)

나머지(파싱·카드 렌더·타임라인 컴파일·ffmpeg MP4 렌더·CapCut/FCPXML 빌드)는 결정적 코드라 AI를 안 쓴다 — 그래서 결과가 매번 똑같고 디버깅이 쉽다.

② 사진 후보 선택 — 이제 전용 picker로 (생략 해제됨)

weft pick — 왼손 단축키 picker

  • 완성 브라우저 picker: shot별 후보를 나란히 보고 고른다. 왼손 단축키 — W/S shot 이동, A/D 후보 이동, Space/1~9 선택, G 추가 생성.
  • AI 추가 생성 picker 안에서 + 생성 → 연결된 gpt-image-1으로 후보를 더 만든다.
  • 프롬프트 수정 프롬프트를 고쳐 저장하거나 그 프롬프트로 바로 재생성.
  • 외부 이미지 드래그&드롭 또는 파일 경로로 직접 넣기.
  • 선택 반영 고른 픽은 PICKS.json에 저장 — weft ffmpeg 재실행이 그대로 반영. CapCut을 쓸 땐 picker의 🎬 CapCut 빌드 버튼으로 draft 재생성.
  • 이전 상태 처음엔 비용 위해 후보 1장(--n 1)이라 "선택"이 없었음 → 지금은 후보 2장 생성, picker로 선택 가능.

③ 사람의 손이 들어가는 곳

🚪 대본 작성·확정

직접 쓴다 — 무엇을·어떤 톤으로 (Weft 외부).

🚪 콘티 검토

홀드/몽타주로 이미지 수 감축 확인.

🚪 보이스 선택

샘플 듣고 1회.

🚪 이미지 픽 (picker)

후보 선택·재생성·외부 삽입 (선택).

🚪 카드 문구

인용·숫자 카드 내용.

🚪 최종 검수·발행

MP4 확인 — 더 다듬을 땐 CapCut/FCPXML 핸드오프.

④ 명령어 (순서대로)

# 실행: 키는 .env (Typecast/OpenAI) · 렌더/화질 옵션은 WEFT_SETTINGS.txt
# 0. 대본 — 사용자가 직접 작성 (Weft 외부, 주제 자유) → SCRIPT.md
# 1. 콘티 — AI의 script-to-conti 스킬로 변환 → CONTI.md
#      (AI가 스킬 위치를 찾을 때:  weft whereisskill)

cd my-video                # CONTI.md가 든 프로젝트 폴더로 이동
weft conti                 # 2·7. 정본화·검증·타임라인 컴파일 (./CONTI.md → ./generated_project/)
weft tts                   # 3. 🤖 TTS
weft images --n 2          # 4. 🤖 이미지(후보 2장)
weft animate               # 5. 🤖 remotion/hyperframe shot이 있을 때만
weft ffmpeg                # 6·8. 카드 렌더·기본 MP4 → EXPORTS/weft_render.mp4 🎬

# 한 번에 (conti → tts → images → animate check → ffmpeg):
weft all
weft settings              # WEFT_SETTINGS.txt 생성/확인 (화질·인코더·export 옵션)

# 선택 — 더 다듬고 싶을 때
weft pick                  # 🖼 picker (후보 선택·재생성·외부 이미지)
weft capcut                # 9. CapCut draft 핸드오프 (CapCut 종료 상태에서)
weft fcpxml                # 9. FCPXML 핸드오프 (Final Cut/Premiere/Resolve)
Weft · 내 대본 → 영상 자동화 (주제 불문) · AI(LLM·TTS·이미지·애니메이션) + 결정적 코드 + 휴먼 터치 게이트 · 기본 산출물은 ffmpeg MP4, CapCut/FCPXML은 선택 핸드오프
